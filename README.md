# hubot-zhihu

hubot知乎日报获取

See [`src/zhihu.coffee`](src/zhihu.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-zhihu --save`

Then add **hubot-zhihu** to your `external-scripts.json`:

```json
["hubot-zhihu"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
